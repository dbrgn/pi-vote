== Building under Linux ==

You can build Pi-Vote using MonoDevelop 2.4 or later. 
Open the PiVote.sln solution file and build. Don't 
forget to install the libgmp3.


== Building under Windows ==

You can use MonoDevelop 2.4 or later or Visual Studio
2010. You can use the libgmp binary or build this
from its own repository.


== Other files needed ==

You can find the other files needed in the build
folder. They include the root certificate, the config
file and the Emil.GMP.dll.config as wells as the
desktop icon for debian/linux.
